# VoxGen — Documentation Index

> AI-powered system-wide dictation and content generation desktop app.
> **Current Version:** v2.16.4 | **Last Scanned:** 2026-02-23

## Project Overview

- **Type:** Multi-part monorepo (Desktop App + API Backend + Marketing Website)
- **Primary Language:** TypeScript 5.7
- **Architecture:** Electron two-process (main + renderer) with provider pattern
- **UI:** React 19 + Tailwind CSS 3.4 + shadcn/ui (Radix)

## Quick Reference

### Desktop App (primary)
- **Entry Point:** `electron/main/index.ts` (main process), `src/App.tsx` (renderer)
- **Tech Stack:** Electron 33 + React 19 + Vite 6 + TypeScript 5.7
- **Build:** `npm run build` → `npm run electron:build:win`
- **Pattern:** Provider factories (STT + Cleanup), React Context for settings

### API Backend
- **Entry Points:** `api/*.ts` (individual serverless functions)
- **Tech Stack:** Vercel serverless + Supabase + Stripe
- **Functions:** 11 of 12 max (Hobby plan limit)

### Marketing Website
- **Entry Point:** `website/index.html`
- **Tech Stack:** Static HTML/CSS
- **Deployment:** Auto-deploy via Vercel on push to master

```bash

# Start development (Vite + Electron with hot reload)

# Build Windows installer
npm run build

```
